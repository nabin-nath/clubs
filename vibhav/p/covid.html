VIBHAV - Automatic Covid Gateway
Images
Description:

Operations:

This project is made to prevent the spread of Covid-19. A device is made which can perform multiple operations such as mask detection, temperature measurement and sanitization using combination of different hardware.

Abstract:

The system is designed to detect the faces and to determine whether the person has worn a face mask or not. Using the above data, we can decide whether the concerned person can be allowed inside public places such as market or a hospital. This project can be used in the hospitals, markets, bus terminals, restaurants, and other public gatherings where the monitoring has to be done. This project consists of a camera that will capture the images of the people entering public places and detect whether the person has worn a face mask . A sanitizing device and automatic door opening technology will also be used.

Execution:

When any person puts his/her hand in front of the device, it checks the body temperature using the temperature sensor. Simultaneously, it checks whether the person is wearing a mask or not using Machine learning. If required conditions are satisfied then only door will open automatically . If required conditions are not satisfied then the device glows a red LED and alerts everyone using buzzer.

Applications:

• This device can be used at the entrance of crowded places such as malls, offices, shops, stadiums, hospitals, markets, bus terminals, restaurants, and other public gatherings .
• This device will be really useful for today's world due to this pandemic situation and will surely help in fighting against the virus.It will control the spread of COVID-19.
• It makes the process of temperature checking contactless and more safe.
• The device is available at an affordable price that makes it suitable for the users.
